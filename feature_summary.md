# Сводка функционала AI Сервера

Сервер состоит из набора Docker контейнеров, каждый из которых выполняет свою специфическую задачу. Основное управление и использование происходит через двух Telegram ботов.

## Telegram Боты

| Бот | Описание | Основные функции |
| :--- | :--- | :--- |
| **🤖 Кодер-Бот**<br/>`tg-coder-bot` | Ваш личный AI-помощник и разработчик. Подключен к Ollama для генерации кода и текста. | • Общение с нейросетью<br/>• Написание кода и рефакторинг<br/>• **Поиск в интернете** (через команду `/search`)<br/>• Удержание контекста диалога |
| **🛠️ Админ-Бот**<br/>`tg-admin-bot` | Бот для управления сервером, мониторинга и администрирования. | • **Управление Ollama**: выбор активной модели и загрузка новых.<br/>• **Логи**: получение логов из контейнеров, настройка выгрузки по расписанию.<br/>• **Мониторинг хостов**: проверка доступности сайтов/IP адресов по списку.<br/>• **Система**: проверка загрузки CPU, RAM, SSD и видеокарт (GPU).<br/>• **Удаленные сервера (SSH)**: добавление удаленных серверов по SSH для просмотра их Docker-логов. |

## Основные Сервисы (Контейнеры)

| Сервис | Порт | Описание |
| :--- | :--- | :--- |
| **Ollama**<br/>`ollama` | `11434` | Локальный движок для запуска нейросетей (LLM). Используется кодер-ботом и OpenWebUI. |
| **OpenWebUI**<br/>`openwebui` | `36486` | Красивый веб-интерфейс (похожий на ChatGPT) для общения с вашими моделями Ollama напрямую через браузер. |
| **Code-Server**<br/>`vscode-ide` | `36490` | Полноценный VS Code в браузере. Вы можете редактировать код проекта прямо на сервере. |
| **Vless Client**<br/>`vless-client`| `1080/1081` | VPN/Proxy клиент, который обеспечивает Кодер-боту (и другим контейнерам) доступ к интернету. |

## Как использовать новые функции

*   **Мониторинг Системы (Нагрузка CPU/GPU)**: В Админ-боте нажмите кнопку **«🖥️ Система»**. Бот в реальном времени опросит сервер и выдаст сводку (включая `nvidia-smi` для видеокарт).
*   **SSH Менеджер логов (Nginx Proxy Manager и др.)**: 
    1. В Админ-боте нажмите кнопку **«🌐 Удаленные сервера»**.
    2. Нажмите **«➕ Добавить сервер»** и отправьте IP, логин и пароль от сервера (например, от `155.212.166.17`).
    3. После добавления просто нажимайте на его IP в списке серверов — бот покажет все Docker-контейнеры на нём и позволит в 1 клик вытянуть их логи.
*   **Поиск в интернете (Кодер-Бот)**: Просто напишите Кодеру команду `/search Ваш запрос`. Он использует прокси-сервер для доступа к сети и найдет ответ.
