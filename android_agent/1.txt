### 1. Сервер: Оркестратор
Создай папку `mkdir -p /root/ponytail` и наполни её скриптами.

**Скрипт `tasks.py` (Управление):**
```python
import subprocess
def phone_exec(cmd):
    subprocess.run(f"ssh -p 2222 localhost 'echo \"{cmd}\" > /sdcard/task.txt'", shell=True)
def get_screenshot(fname="screen.png"):
    phone_exec("screencap -p /sdcard/screen.png")
    subprocess.run(f"scp -P 2222 localhost:/sdcard/screen.png {fname}", shell=True)
```

**Скрипт `stt.py` (Голос):**
```python
import whisper
model = whisper.load_model("base")
def transcribe(path):
    return model.transcribe(path, language="ru")["text"]
```

### 2. Android: Агент
Создай файл `/data/data/com.termux/files/home/ponytail/agent.py`:

```python
import os, subprocess, time
def run_cmd(cmd):
    # Авто-выбор: Root или ADB
    prefix = "su -c" if os.path.exists("/system/xbin/su") else "adb shell"
    subprocess.run(f"{prefix} 'input keyevent 26'", shell=True) # Пробуждение
    return subprocess.check_output(f"{prefix} '{cmd}'", shell=True)

while True:
    if os.path.exists("/sdcard/task.txt"):
        with open("/sdcard/task.txt", "r") as f:
            cmd = f.read().strip()
        if cmd:
            out = run_cmd(cmd)
            with open("/sdcard/result.txt", "wb") as f: f.write(out)
            os.remove("/sdcard/task.txt")
    time.sleep(1)
```
